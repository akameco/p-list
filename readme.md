# p-list [![Build Status](https://travis-ci.org/akameco/p-list.svg?branch=master)](https://travis-ci.org/akameco/p-list)

> show dependencis from package.json


## CLI

```
$ npm install --global p-list
```

```
$ p-list --help

  show dependencis from package.json

  Usage
    $ p-list

  Options
    --save,-S       [Default: true]
    --save-dev,-D   [Default: false]

  Examples
    $ p-list
    meow read-pkg-up
    $ p-list -D
    xo ava
```
